This is a complex multi-tenant real-time fleet monitoring and management web and mobile application. It uses data transmitted to a database from devices in the cars. The application has very rich feature set. Some of the more interesting features include:

  • Convenient CRUD
  • Real-time tracking
  • Statistics
  • Analytics giving information about how the cars are used, driving style, fuel costs, etc.
  • Alarms based on:
    • Date
    • Location
    • Sensor data

We used Spring Framework and React.js to deliver performance, scalability and security.
