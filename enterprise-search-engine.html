Javascript, HTML, CSS, Python, React.js, Node.js, CI/CD, GitLab, ElasticSearch

  1. DigidWorks
  2. Projects
  3. Enterprise search, archiving and tagging engine

This is a web-based enterprise search engine that can crawl and index websites and/or documents. After data enters the index it can be searched using full text search and filters. There is also a text tagging module used for adding text annotations to enhance searching functionality.

Some of the features include:

  • Easy to use UI
  • Automated indexing
  • Advanced search filters
  • Auth and roles
  • Text tagging for advanced search

If you are looking for an enterprise search engine, that will help you better organize documents and search your company data in a sane and efficient way, go ahead and write us a few lines about it, we can help!
