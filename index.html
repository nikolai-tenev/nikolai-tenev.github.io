A bit more on the technical side...

We develop enterprise and business management web applications - mostly complex admin dashboards and different management and monitoring enterprise systems.

We use performant technologies, that scale easily, and a "sane" approach, meaning we use the most functional and adequate tools for the project at hand. When needed we use micro services architecture or we even go serverless.

On the UI/UX side of things, we always try to create very practical and easy to use applications. The interfaces we build are responsive and developed with mobile-first approach in mind. We are not afraid of creating elaborate widgets and screens if that is going to make things easier for the end users of the application.
